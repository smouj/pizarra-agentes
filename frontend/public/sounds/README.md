# MGS Codec Sounds

Place your MGS sound files here:

- `beep.mp3` or `beep.wav` - Button press / interface beep sound
- `call-start.mp3` or `call-start.wav` - When initiating a codec call
- `message-received.mp3` or `message-received.wav` - When receiving a message from agent
- `alert.mp3` or `alert.wav` - Error or warning sound

The app will automatically detect and use .mp3, .wav, or .ogg formats.

## Recommended Sources:
- Metal Gear Solid sound effects from game files
- Create simple beep sounds using audio editing software
- Use royalty-free sound effect libraries

## Format:
- Sample rate: 44100 Hz
- Channels: Mono or Stereo
- Bit rate: 128-192 kbps for MP3
