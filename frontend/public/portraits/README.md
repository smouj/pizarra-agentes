# MGS Codec Portraits

Place your 8 agent portrait images here as PNG files (recommended: 400x450px or similar aspect ratio 8:9).

## Required Portrait Files:

1. `orchestrator.png` - Orchestrator agent portrait
2. `shell-executor.png` - Shell Executor agent portrait
3. `browser-agent.png` - Browser Agent portrait
4. `file-manager.png` - File Manager agent portrait
5. `communications.png` - Communications agent portrait
6. `custom-skill.png` - Custom Skill agent portrait
7. `operator.png` - Operator/Gateway portrait (user side)
8. `snake.png` - Snake/Special agent portrait (bonus)

## Image Guidelines:
- Format: PNG with transparency
- Recommended size: 400x450px (8:9 aspect ratio)
- Style: MGS character portraits, green tinted or monochrome
- Background: Transparent or solid dark color
- Keep file sizes reasonable (<500KB per image)

The portraits will be displayed with:
- Green border and glow effect
- CRT scan lines overlay
- Slight brightness/contrast filter
- MGS authentic aesthetic
